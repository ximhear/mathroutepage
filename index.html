MathRoute

중고등학교 수학 공식을 쉽게 찾고 학습할 수 있는 iOS 앱

앱 소개

MathRoute는 중학교와 고등학교 수학 공식을 쉽게 찾고 학습할 수 있는 iOS 앱입니다. LaTeX로 아름답게 렌더링된 수학 공식과 상세한 설명, 예제 문제를 통해 효과적인 수학 학습을 지원합니다.

주요 기능

  • 중학교/고등학교 수준별 공식 제공
  • 대수, 기하, 미적분, 확률통계 등 카테고리별 분류
  • LaTeX로 표현된 아름다운 수식 렌더링
  • 훑어보기 기능으로 빠른 공식 확인
  • 검색 기능으로 원하는 공식 빠르게 찾기
  • 즐겨찾기 및 최근 본 공식 관리
  • 공식의 유도 과정과 예제 문제 제공
  • 오프라인 지원으로 언제든지 학습 가능

자주 묻는 질문 (FAQ)

앱을 처음 실행할 때 로딩이 오래 걸려요.
처음 앱을 실행할 때는 수학 공식 데이터를 다운로드하고 초기화하는 과정이 있어 시간이 소요될 수 있습니다. 안정적인 인터넷 연결 상태에서 기다려주시면 완료됩니다. 이후 사용 시에는 오프라인에서도 빠르게 실행됩니다.
특정 공식을 찾을 수 없어요.
공식은 지속적으로 추가되고 있습니다. 원하는 공식이 없다면 다음 방법을 시도해보세요:
1. 검색 기능을 사용하여 다른 키워드로 검색
2. 관련 카테고리를 훑어보기로 확인
3. 앱 업데이트 확인 (새로운 공식이 추가되었을 수 있음)
없는 공식이 있다면 문의를 통해 요청해주세요.
즐겨찾기한 공식이 사라졌어요.
즐겨찾기 정보는 기기에 저장되며, 다음 상황에서 초기화될 수 있습니다:
1. 앱 삭제 후 재설치
2. iOS 업데이트나 기기 초기화
3. 앱 데이터 초기화
현재는 클라우드 동기화를 지원하지 않으므로 중요한 공식은 따로 기록해두시는 것을 권장합니다.
앱이 업데이트되지 않아요.
MathRoute는 서버에서 자동으로 새로운 공식과 업데이트를 확인합니다. 업데이트가 되지 않는 경우:
1. 인터넷 연결 상태 확인
2. 앱을 완전히 종료한 후 다시 실행
3. 홈 화면에서 아래로 당겨서 새로고침
4. App Store에서 앱 업데이트 확인
오프라인에서도 사용할 수 있나요?
네, 가능합니다! 한 번 다운로드된 공식은 오프라인에서도 확인할 수 있습니다. 단, 새로운 공식 업데이트나 동기화는 인터넷 연결이 필요합니다.
